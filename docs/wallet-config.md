# Wallet creation and configuration

```bash
cleos --wallet-url http://localhost:8888 wallet create -n eosmechatero
cleos -u https://jungle.eosio.cr --wallet-url http://localhost:8888 wallet list
cleos -u https://jungle.eosio.cr --wallet-url http://localhost:8888 wallet unlock -n eosmechatero
cleos -u https://jungle.eosio.cr --wallet-url http://localhost:8888 wallet import -n eosmechatero
```bash
